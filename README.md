# Dialog Manager
------------------------------------------------------------------------

Support JAVA 7

### Gradle:

Step 1. Add it in your root build.gradle at the end of repositories:

```gradle
allprojects {
	repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```

Step 2. Add the dependency
```gradle
dependencies {
	        compile 'com.github.daolq3012:DialogManager:0.1'
}
```
